## Зависимости
`Python 3`

## ВНИМАНИЕ!
**Не добавляйте слишком много фраз и не запускайте скрипт много раз подряд! У вас будет ошибка о капче!**

## Использование

Заполните `msgs.mudr` и `tokens.mudr` (как их заполнять указано там же)

Запустите скрипт `mudro.py`
```bash
python3 mudro.py
```
